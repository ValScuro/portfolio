### ✨ About the Project
This is a modern, responsive, and interactive portfolio built with **React**, **Chakra UI**, and **Framer Motion**. It features:
- 📱 Fully responsive design
- 🌓 Dark/Light mode
- ✨ Smooth animations
- 📁 Project showcase
- 🥚 Hidden Easter eggs

### 🛠 Tech Stack
- **Frontend:** React 19, Vite, Chakra UI
- **Animations:** Framer Motion
- **Deployment:** GitHub Pages
